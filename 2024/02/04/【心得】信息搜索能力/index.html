【心得】信息搜索能力

一、搜索技术的重要性

  1. 搜索技术是冲浪必备技能。
  2. 掌握搜索技术是互联网时代获取信息的关键,可以极大程度地缩小信息差距。

二、搜索的目的

  1. 了解信息(Know Something)
  2. 学习技能(Learn Something)
  3. 创作内容(Create Something)
  4. 完成任务(Do Something)

三、搜索的内容

  1. 信息资讯:新闻、事件等
  2. 知识技能:概念、教程等
  3. 素材文件:图片、视频、文档等
  4. 工具软件:各类工具、插件等

四、搜索的原则和规则

  1. 要快速准确,谷歌效率最高。
  2. 可运用搜索语法提升效率:
    1. 限定关键词:用双引号””包裹
    2. 限定标题:intitle
    3. 限定内容:intext
    4. 限定网址:inurl
    5. 限定网站:site
    6. 限定图片大小:imagesize
    7. 限定文件类型:filetype
  3. 信息源头最准确可信,要注意判断来源。

五、搜索知识技能类

  1. 主要平台:谷歌、得到APP、优质视频网站等
  2. 注意知识质量,尽量采用一手资料。
  3. 可下载研究报告、学术论文等高质量文档。
  4. 视频教程可在视频网站、短视频平台获取。

六、搜索素材文件类

  1. 可利用免费素材网站、版权网站等。
  2. 注意遵守版权,非商用基本没有风险。
  3. 可用各种下载工具下载素材。
  4. 需注意素材的质量、风格等符合要求。

七、搜索工具软件类

  1. 优先考虑在线工具。
  2. 其次是软件,最后是插件。
  3. 大多数需求都能找到对应的工具。
  4. 可用类似网站找相似工具。

八、运用AI工具

  1. ChatGPT等AI工具可直接回答问题,无需搜索。
  2. 可进行追问、对比、推理等,功能强大。
  3. 结合传统搜索技术和AI工具,形成超级搜索技术。
  4. AI将深刻影响搜索方式,要积极运用。
