这是黑马2023前端Web开发前131集的阶段练习，内容为小兔鲜网站的首页：展示地址 / 源代码地址

本次练习的特点有：素材划分更细、使用字体图标、SEO优化等

项目目录

  • images：存放固定使用的图片素材
  • uploads：存放非固定使用的图片素材
  • iconfont：字体图标素材
  • css
    • base.css：基础公共样式
    • common.css：各个网页相同模块的重复样式
    • index.css：首页样式
  • index.html：首页HTML文件

SEO三大标签

SEO定义：搜索引擎优化，提升网站百度搜索排名

提升SEO的常见方法：

  • 竞价排名
  • 将网页制作成html后缀
  • 标签语义化（在合适的地方使用合适的标签）
  • ……

网页头部SEO标签：

  • title
  • description
  • keywords

Favicon图标

定义：网页图标，出现在浏览器标题栏，增加网站辨识度

图标：favicon.ico，一般存放在网站的根目录里面

版心

wrapper宽度：1240px

快捷导航

结构：通栏>版心>导航ul

布局：flex-end

头部导航

结构：.header>logo+nav+search+cart

底部区域

结构：通栏>版心>服务+帮助+版权

由于是底部区域，不用考虑SEO优化（即不用li标签包裹a标签）

banner

结构：通栏>版心>轮播图+侧导航+原点指示器

新鲜好物区域

结构：标题+内容

由于格式重复，使用公共样式

热门品牌区域

标题结构：左侧+右侧箭头

生鲜区域

标题结构：右侧>菜单+查看全部
