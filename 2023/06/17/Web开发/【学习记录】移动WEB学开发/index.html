平面转换

属性名：transform

作用：为元素添加动态效果，一般与过渡transition配合使用

概念：改变盒子在平面内的形态（位移、旋转、缩放、倾斜）

平面转换又叫2D转换

平移

属性：transform: translate(X轴移动距离,Y轴移动距离)

取值：

  • 像素单位数值
  • 百分比（参照盒子自身尺寸计算结果）
  • 正负均可

技巧：

  • translate()只写一个值，表示沿着X轴移动
  • 单独设置某个轴，可以使用translateX() / translateY()

旋转

属性：transform: rotate(旋转角度)

角度单位：deg

技巧：

  • 正负均可
  • 取正代表顺时针；反之同理

缩放

属性：

  • transform: scale(缩放倍数)
  • transform: scale(X轴缩放倍数, Y轴缩放倍数)

技巧：

  • 通常，只为scale()设置一个值，表示X轴和Y轴等比例缩放
  • 取值大于1表示放大，取值小于1表示缩小

给width height添加过渡变化，达不到从中心等比例缩放的效果

倾斜

属性：transform: skew(角度度数)

渐变

渐变是多个颜色逐渐变化的效果，一般用于设置盒子背景

分类：

  • 线性渐变
  • 径向渐变

线性渐变：

属性：background-image: linear-gradient(渐变方向, 颜色1 终点位置, 颜色2 终点位置, ...)

取值：

  • 渐变方向：可选
    • to 方位名词
    • 角度度数
  • 终点位置：可选
    • 百分比

径向渐变：

属性：background-image: radial-gradient(半径 at 圆心位置, 颜色1 终点位置, 颜色2 终点位置, ...)

取值：

  • 半径可以是2个，代表椭圆
  • 圆心位置取值：像素单位数值 / 百分比 / 方位名词

常用场景：给按钮添加高光效果

特殊转换

改变转换原点

默认情况下，转换原点是盒子中心点

属性：transform-origin: 水平原点位置 垂直原点位置

取值：

  • 方位名词（left top right bottom center）
  • 像素单位数值
  • 百分比

多重转换

旋转会改变坐标轴向

多重转换时，以第一种转换形态的坐标轴为准

具有层叠性，以最后写的transform属性为准

类似轮子在平地滚动的效果：先平移再旋转–transform: translate() rotate()

空间转换

空间：是从坐标轴角度定义的X、Y和Z三条坐标轴构成了一个立体空间，Z轴位置与视线方向相同

平移

属性名：transform:

属性值：

  • translate3d(x,y,z)
  • translateX()
  • translateY()
  • translateZ()

视距

作用：指定了观察者与z=0平面的距离，为元素添加透视效果（近大远小、近实远虚）

属性名：perspective

属性值：（添加给直接父级，取值范围800px-1200px）

translate3d()必须有三个值，否则不生效

空间转换也叫3D转换

电脑默认是平面，无法观察 Z 轴平移效果，没有则可以不设置视距

旋转

属性：

  • transform: rotateX(角度度数)

  • transform: rotateY(角度度数)

  • transform: rotateZ(角度度数)

  • transform: rotate3d(x, y, z, 角度读书)：用来设置自定义旋转轴的位置即旋转的角度 ，x, y, z 属于 [0~1]

左手法则 - 根据旋转方向确定取值正负

立体呈现

作用：设置元素的子元素是位于**3D**空间还是平面中

属性名：transform-style

属性值：（添加给直接父级）

  • flat（子级处于平面中）
  • preserve-3d（子级处于空间中）

使用步骤：

  1. 直接父元素添加transform-style: preserve-3d
  2. 子级添加position定位，并统一参照位置
  3. 调整子盒子的位置（位移或平移）

立方体的每个面都有独立的坐标轴，互不影响

缩放

作用：让3D盒子本身进行缩放

属性名：transform

属性值：

  • scaleX(值)
  • scaleY(值)
  • scaleZ(值)
  • scale3d(值, 值, 值)

动画

定义：实现多个状态间的变化过程，且过程可控（重复播放、最终画面、是否暂停）

属性名：animation

属性值：动画名称 动画时长 速度曲线 延迟时间 重复次数 动画方向 执行完毕时状态（复合属性）

注意：

  • 动画名称和动画时长必须赋值
  • 取值不分先后顺序
  • 如果有两个时间值，第一个时间值表示动画时长，第二个时间表示延迟时间

速度曲线：

  • linear（匀速）
  • steps(整数)（分步/逐帧动画–配合精灵图实现精灵动画）
  • 加速、减速等不常用

延迟时间：

  • 数字+s（表示动画不会立即开始，延迟一定时间后开始）

重复次数：

  • 整数（表示动画的重放播放次数）
  • infinite（表示无限循环播放）

动画方向：

  • alternate（表示反向播放动画–从最终状态开始，到开始状态结束）

执行完毕时状态：

  • forwards（完毕时停在最终状态）
  • backwards（完毕时停在开始状态–默认）

实现步骤：

  1. 定义动画（两种方式）：

    • @keyframes 动画名称 {
    form {}
    to {}
}

  2. 使用动画

    animation: 动画名称 动画持续时长;

拆分属性：

动画属性支持同时设置多组，用逗号隔开即可

动画属性支持省略动画初始状态（即to、0%的部分代码）

移动适配

分辨率分为硬件分辨率和逻辑分辨率，做适配是以逻辑分辨率（设备屏幕缩放后的分辨率）为基础

视口

作用：显示HTML网页的区域，用来约束HTML尺寸

标签：<meta name="viewport" content="width=device-width, initial-scale=1.0">

属性值：

  • width=device-width（表示视口宽度 = 设备宽度）
  • initial-scale=1.0（不缩放）

手机屏幕尺寸不同，网页宽度均为100%；网页的宽度和逻辑分辨率的尺寸相同

二倍图

概念：设计稿里面每个元素的尺寸的倍数

作用：防止图片在高分辨率屏幕下模糊失真

现阶段设计稿**参考iPhone6/7/8，设备宽度375px**产出设计稿，二倍图设计稿尺寸：750px

使用二倍图时，需要将设计稿切换到2x模式

适配方案

宽度适配：宽度自适应

  • 百分比布局
  • Flex布局

等比适配：宽高等比缩放

  • rem
  • vm

百分比布局、flex布局只能做到宽度适配，适合PC端；移动端适配则使用rem或vm

rem适配方案

定义：rem是相对单位，相对于**HTML标签的字号**计算结果，1rem=1HTML字号大小

使用步骤：

  1. 设置HTML标签字号大小
  2. 使用rem给标签设置尺寸

HTML字号如何设置：

  • 考虑到手机屏幕大小不同、分辨率不同、设备宽度不同，HTML标签字号不应该设置为**px绝对尺寸**，否则没有适配效果

  • 媒体查询能够检测视口的宽度，然后编写差异化的CSS样式，当某个条件成立，执行对应的CSS样式

  • 目前rem布局方案中，将网页等分为10份，HTML标签的字号为视口宽度的1/10（重要）

**flexible.js**：

定义：它是手淘开发的一个用来适配移动端的js库

核心原理：根据不同的视口宽度给网页中html根节点设置不同的font-size

rem单位尺寸：

  1. 确定基准根字号
    • 查看设计稿宽度 → 确定参考设备宽度（视口宽度） → 确定基准根字号（1/10视口宽度）
  2. rem单位尺寸
    • rem单位的尺寸 = px单位数值 / 基准根字号

Less

定义：它是一个CSS预处理器，less文件后缀是.less。

作用：扩充了CSS语言，使CSS具备一定的逻辑性、计算能力

注意：浏览器不识别Less代码，目前阶段，网页要引入对应的CSS文件

VS Code 插件：Easy LESS，保存less文件后自动生成对应的CSS文件

注释

单行：//（ctrl+/）

块：/* */（shift + alt + a）

单行注释无法转换到对应的css文件中

运算

加减乘除直接书写计算表达式，除法需要添加小括号或.–width: (100 / 4px); height: 100 ./ 4px

如果计算表达式中有多个单位，转换为CSS代码时以最后一个单位为准

嵌套

作用：快速生成后代选择器

转换为CSS后如下：

嵌套但**不生成后代&**：

用法：&表示当前选择器（写到谁的大括号中就表示谁），并且不会生成后代选择器

变量

概念：容器，存储数据

作用：存储数据，方便使用和修改

语法：

  • 定义变量：@变量名: 数据;

  • 使用变量：CSS属性: @变量名;

导入

作用：导入less公共样式文件

语法： @import "文件路径";

提示：如果是less文件可以省略后缀

导出

作用：将当前less文件导出为指定css名称和存储路径

写法：在less文件的第一行添加// out: 存储URL

文件夹名称后面添加/

禁止导出：// out: false

vw适配方案

定义：它时一种相对单位，相对视口的尺寸计算结果，1vw = 1/100视口宽度

vm单位尺寸：

  1. 确定设计稿对应的vm尺寸（1/100视口宽度或高度）
    • 一般是查看设计稿宽度 → 确定参考设备宽度（视口宽度） → 确定vm尺寸
  2. vm单位尺寸 = px单位数值 / （1/100视口宽度或高度）

拓展：和vw对应有1vh = 1/100视口高度，但是不能将两者混用，否则某些场景（比如全面屏）导致盒子变形

区别：vm不需要搭配媒体查询使用；vm相当于百分率机制进行适配，而rem默认十分制

响应式网页

定义：一套代码适配所有屏幕设备

业界主流方案：

  • 媒体查询
  • Bootstrap

媒体查询

作用：根据条件是否成立，差异化设置CSS属性

完整写法

@media 关键词 媒体类型 and (媒体特性) { CSS代码 }

可以简写为@media (媒体类型:媒体特性) { CSS代码 }

关键词/逻辑操作符

  • and
  • only
  • not

媒体类型

定义用来区分设备类型的，如屏幕设备、打印设备，其中手机、电脑、平板都属于屏幕设备

媒体特性

定义：用来描述媒体类型的具体特征，如当前屏幕的宽高、分辨率、横竖屏等

  • max-width（最大宽度）
  • min-width（最小宽度）
  • width（宽度）
  • height（高度）
  • orientation
    • portrait（竖屏）
    • landscape（横屏）

书写顺序要考虑CSS属性的层叠性

外部CSS

媒体查询可以配合单独CSS文件使用，只需要在link标签内设置media属性，如下：

<link rel="stylesheet" href="xx.css" media="逻辑符 媒体类型 and (媒体特性)>"

可以简写为<link rel="stylesheet" href="xx.css" media="(媒体类型:媒体特性)>"

Bootstrap

由Twitter开发维护的前端UI框架，提供了大量编写好的CSS，允许用户结合一定HTML / JavaScript，快速开发网页，附Bootstrap中文网

栅格系统：将网页宽度分为12等份，每个盒子占用的对应的份数，以下是bootstrap的响应式断点（共6个响应区间），需要配合版心class=container使用

区分名 缩写 维度
Extra small None <576px
Small se ``>=576px`
Medium md ``>=768px`
Large lg ``>=992px`
Extra large xl ``>=1200px`
Extra extra large xxl ``>=1400px`
