Hello, HTML!

HTML超文本标记语言–HyperText Markup Language，超文本指链接，标记则是带尖括号的文本，尽管不是编程语言，但是这样的性质已经能实现网页展示、跳转从而形成网络。

html 1.0发布于1991年，html 5发布于2008年，现在仍处于蓬勃的发展时期。

规范的网页需要符合既定的标准，即HTML的基本骨架

可以得出，HTML基本骨架包含如下标签：

  • html–整个网页
  • head–网页头部
  • title–网页标题
  • body–网页主体

在head标签中还可以添加如下标签：

meta标签提供有关HTML文档的元数据，例如字符集、页面有效期、页面作者、关键字列表、页面描述等信息

base标签为页面中所有相对链接指定一个基本链接

link标签常用于引用外部 CSS 样式表

style标签在当前文档中内嵌CSS样式

script标签在当前文档定义JavaScript 脚本

noscript标签定义一些内容来替代不能运行的 JavaScript 脚本或者给用户一些提示

标签的关系

明确标签的书写位置，即相对关系，可以使结构更加整齐

  • 父子关系（嵌套）
  • 兄弟关系（并列）

PS: 注意合理使用缩进和对齐，让标签关系一目了然

标题标签

标签名：h1 / h2 / h3 / h4 / h5 / h6（双标签）

显示特点：

  • 文字加粗
  • 字号逐级递减
  • 独占一行

PS: h1标签在一个网页只能使用一次，用来放新闻标题或网页的logo

段落标签

标签名：p（双标签）

显示特点：

  • 独占一行
  • 段落之间存在间隙

换行标签

标签名：<br>（单标签）

PS: 浏览器不会识别Enter回车键的换行

水平线标签

标签名：<hr>（单标签）

文本格式化标签

作用：为文本添加特殊模式，突出重点。包括加粗、倾斜、下划线、删除线

标签名：

  • 加粗：strong / b
  • 倾斜：em / i
  • 下划线：ins / u
  • 删除线：del / s

显示特点：在一行显示

PS: strong表示该文本重要且需要突出，b则表示这需要加粗，两者的机制稍有不同，推荐使用前者，其他3个标签同理

图像标签

作用：在网页中插入图片

标签名：img（单标签）

属性：

  • src（必须属性，用于指定图像来源位置）
  • title（图片名字）
  • alt（替换文本–图片无法显示的时候显示的文字）
  • width（图片的宽度–值为数字且没有单位）
  • height（图片的高度–值为数字且没有单位）

超链接标签

作用：点击跳转到其他页面

标签名：a（双标签）

属性：

  • href（必须属性，跳转的链接）
  • target（设置为_blank表示打开新页面跳转）

PS: 将href设置为#表示空连接，适合开发初期不确定正式链接时使用

音频标签

标签名：audio（双标签）

属性：

  • src（必须属性，音频URL）
  • controls（显示音频控制面板）
  • loop（循环播放）
  • autoplay（自动播放）

视频标签

标签名：video（双标签）

属性：

  • src（必须属性，视频URL）
  • controls
  • loop
  • muted（静音播放）
  • autoplay（支持在静音模式下自动播放）

列表标签

内容：布局内容排列整齐的区域

标签名：

  • ul（无序列表）、li（列表条目）–ul嵌套li
  • ol（有序列表）、li（列表条目）–ol嵌套li
  • dl（定义列表）、dt（定义列表的标题）、dd（定义列表的描述 / 详情）–dl嵌套dt和dd

ul、ol标签里面只能包含li标签，li标签可以包含任何内容

dl标签里面只能包含dt和dd标签，dt和dd标签可以包含任何内容

表格标签

作用：类似Excel表格，用来展示数据

标签名：table（表格）、tr（行）、th（表头单元格）、td（内容单元格）–table嵌套tr，tr嵌套th / td

属性：

  • border（添加边框线，值为线的宽度）

表格结构标签：用于让浏览器清楚表格的结构，了解即可

  • thead（表格头部）
  • tbody（表格主体）
  • tfoot（表格底部）

合并单元格：将多个单元格合并为一个单元格

  • 跨行合并：确定目标区域，保留最上单元格，添加属性rowspan（值为需要合并的单元格数量），删除其它单元格
  • 跨列合并：确定目标区域，保留最左单元格，添加属性colspan（值为需要合并的单元格数量），删除其它单元格

表头单元格默认加粗居中

表格结构标签嵌套对应的tr标签做标识的作用，告诉浏览器表格结构，对开发过程用处不大，了解即可

合并单元格不能跨表格结构合并

表单标签

作用：收集用户信息

input标签

标签名：input（单标签）

属性：

  • type（功能）

    • text（文本框用于输入单行文本）

    • password（密码框）

    • radio（单选框）

    • checkbox（多选框）

    • file（上传文件）

  • placeholder（占位文本–用于文本默认显示）

  • name（控件名称–用于分组单选框，同组只能选中一个）

  • checked（默认选中）

  • multiple（文件多选–让file功能可以选中多个文件）

type属性值不同，则功能不同

placeholde对能显示文本的功能适用

下拉菜单

标签名：select（下拉菜单）、option（菜单项）–select嵌套option

属性：

  • selected（默认选中）

PS: 文本域右下角有拖拽缩放功能，但一般会禁用

文本域

作用：多行输入文本的表单控件

标签名：textarea（双标签）

属性：

  • ``

label标签

作用：网页中某个标签的说明文本

用途之一：绑定文字和表单控件的关系，增大表单控件的点击范围

按钮

标签名：button（双标签）

属性：

  • type（按钮类型）
    • submit（提交按钮–点击后可以提交数据到后台）
    • reset（重置按钮–点击后将表单控件恢复默认值）
    • button（普通按钮–默认没有功能，一般配合JavaScript使用）

PS: 这些功能只在form标签对应的同一表单区域有效

表单

标签名：form

属性：

  • method（提交方式–可选POST / GET）
  • action（提交目标地址）

div 和 span

它们属于 无语义 标签

作用：布局网页（划分网页区域，摆放内容）

显示特点：

  • div独占一行
  • span不换行

传统布局一般就是各种设置标签在网页上的位置，html5专门提出了多个语义更明确的“布局类”标签，比如header nav section article aside footer details summary

字符实体

作用：在网页中显示预留字符

常用字符实体：

显示字符 说明 实体字符
空格 &nbsp;
< 小于号 &lt;
